JAVASCRIPT

¿Que es JavaScript?

JavaScript se ha diseñado para añadir interactividad a las páginas HTML

JavaScript es un lenguaje de scripting

Un lenguaje de scripting es un lenguaje de programación ligero JavaScript no está por lo general integrado directamente en páginas HTML

JavaScript es un lenguaje interpretado, es decir, no necesita compilarse para ejecutarse

Objetivo Finalidad
Especificar una librería para uso de tareas comunes principalmente para el desarrollo fuera del navegador web. Controlar el navegador y añadir dinamismo e interactividad a las páginas Web.

HISTORIA

1995

Brendan Eich desarrolla liveScript para incluirlo en el netscape navigator 2.0 poco tiempo despues pasa a llamarse javaScript

1996

Microsoft Crea jScript para competir con Javascript y evitar problema de licencia

1997

Se envia la version de javaScript 1.1 a la ECMA, para una estandarizacion internacional. Llamada ECMAScript

Ventajas Desventajas
Es rápido, por lo tanto tiende a ejecutar las funciones inmediatamente. En el FrontEnd sus códigos son visibles, por lo tanto pueden ser leídos por cualquier usuario.
Cuenta con múltiples opciones de efectos visuales. Tiende a introducir gran cantidad de fragmentos de código en los sitios web.
Es muy versátil, puesto que es muy útil para desarrollar páginas dinámicas y aplicaciones web. No es compatible en todos los navegadores de manera uniforme.
Es multiplataforma, puede ser ejecutado de manera híbrida en cualquier sistema operativo móvil. Los usuarios tienen la opción de desactivar JavaScript desde su navegador.
